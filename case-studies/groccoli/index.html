Groccoli

2026-07-21Tauri 2React 19TypeScriptRustZustand

Problem
AI work can fracture between conversation, planning, local tools, and build execution. The system needs to preserve user intent while keeping powerful actions scoped and visible.
Solution
Groccoli is a private desktop workspace that unifies Grok-oriented Chat and Build flows in one interface, with a Friday intent layer, modular capability pedals, MCP-shaped local tools, and scoped execution boundaries.
Current evidence
Version 0.4.0 exists as a local desktop build. In this review session, the Friday smoke suite, frontend production build, and Rust backend check passed.
Scope
The source tree remains private. The public walkthrough focuses on product role, stack, verification status, and design/architecture choices.

Why It Matters

Groccoli shows the bridge between my creative-director background and systems engineering: the interface is not decoration layered on top of the engine. The interface is how the engine becomes legible, controllable, and trustworthy.

The product pattern is a focused desktop cockpit: one workspace, two operating modes, stable goal memory, permission-aware tools, and visible boundaries around what the agent can touch.

What Was Built

  • A Tauri desktop app shell with React, TypeScript, Vite, Zustand state, and Rust backend configuration.
  • A Chat to Build workspace model that keeps a canonical user directive active across mode changes.
  • A modular capability layer for intent orchestration, local tools, project memory, token visibility, scoped file/git reads, and parallelizable MCP-style tasks.
  • A disciplined evidence posture: implementation can be walked through privately while the public page stays focused on verifiable product and architecture claims.

Current Evidence

  • Smoke verificationFriday directive preservation and build handoff smoke suite passed after updating a stale UI-wiring assertion to match the current hook-based rail implementation.
  • Frontend buildTypeScript project build and Vite production build completed successfully.
  • Backend checkRust backend check completed successfully for the Tauri source.
